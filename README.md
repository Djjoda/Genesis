## SciencePayload from Genesis 1:11 TM

SciencePayload is built on Ionic Framework 5.2.3

##Installation

1. git clone https://github.com/Djjoda/Genesis.git

2. cd Genesis

3. npm install

4. ionic serve --devapp

The main encoding is in ../src/app/tab1/tab1.page.ts
